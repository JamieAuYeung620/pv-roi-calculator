# Non-expert homeowner usability test (template)

## 5-minute test script
1. Ask participant to think aloud while using the app.
2. “Set up a typical home scenario and run a calculation.”
3. “Find where yearly savings and lifetime value are shown.”
4. “Switch tariff mode between A, B, and Compare, and explain the difference.”
5. “Change PV system size (kWp) and describe what that means in your own words.”
6. “Open one chart and explain what it tells you.”
7. “Point to any input that feels unclear or risky to set incorrectly.”

## Targeted questions (confusion-focused)
1. What do `Tariff A`, `Tariff B`, and `Compare` mean to you?
2. Was `kWp` clear? If not, what wording would be clearer?
3. Did you understand what `discount rate` changes in results?
4. Which chart was easiest to understand, and why?
5. Which chart was hardest to understand, and why?
6. Did any label sound too technical or ambiguous?
7. Did any value look important but unexplained?
8. What one wording change would most improve trust or clarity?
